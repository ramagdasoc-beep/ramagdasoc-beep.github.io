# CMSC 207 Mini-Project: Build a Personal Web Profile

## Overview
This project is a personal web profile for Regine A. Magdasoc, created as a mini-project for CMSC 207. It showcases a modern, responsive web design using HTML, CSS, and JavaScript. The site highlights professional background, interests, and contact information, serving as a digital introduction and portfolio.

## Features
- **Responsive Design:** Adapts seamlessly to mobile and desktop screens.
- **Custom Theme:** Uses a palette inspired by personal branding.
- **Profile Section:** Displays a photo, tagline, and biography.
- **Hobbies & Interests:** Cards for family, reading, and K-Dramas.
- **Contact Section:** Email and social media links.
- **Interactive Teaching Style Button:** Reveals insights about teaching style with each click.

## File Structure
- `index.html` — Main webpage structure and content.
- `style.css` — Custom styles, theme, and responsive layout.
- `script.js` — Handles button interaction for teaching style insights.
- `assets/img/` — Profile and related images.

## Usage
Open `index.html` in any web browser. The site is fully static and requires no backend.

## How the Interactive Button Works
The "My Teaching Style" button cycles through several insights about Regine's teaching approach. Each click updates the displayed text, making the profile more engaging.

## Setup Instructions
1. **Clone the repository:**
   ```bash

   ```

   ```bash
   cd ramagdasoc-beep.github.io
   ```
3. **Open `index.html` in your browser:**
   - Double-click `index.html` or open it with your preferred web browser.

No additional installation is required. All files are static and ready to use.
